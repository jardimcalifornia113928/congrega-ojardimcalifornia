# Documento 04 - Implementação para o Gemini

## Instruções obrigatórias

-   Não criar novas tabelas.
-   Não alterar modelos existentes.
-   Não modificar APIs.
-   Não refatorar componentes existentes.
-   Criar apenas o módulo de impressão.

## Preview

O botão 'Visualizar Impressão' deve abrir uma prévia.

## Exportação

Permitir: - Salvar PDF - Imprimir

## Critérios de aceite

-   Dados iguais aos exibidos nas telas.
-   Respeitar seleções.
-   Incluir fim de semana na mesma impressão.
-   Layout fiel ao PDF.
