# Documento 01 - Visão Geral

## Objetivo

Implementar o módulo de impressão utilizando o layout PDF fornecido como
referência visual.

## Escopo

-   Adicionar apenas a funcionalidade de impressão.
-   Não alterar regras existentes.
-   Não modificar banco de dados.
-   Não modificar APIs.
-   Não alterar a lógica das escalas.

## Fluxo

1.  Tela Meio de Semana.
2.  Tela Fim de Semana.
3.  Botão Visualizar Impressão.
4.  Preview.
5.  Salvar em PDF ou Imprimir.

## Restrições

A impressão deve refletir exatamente o estado das telas.
